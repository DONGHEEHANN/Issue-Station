# 🚅이슈역(Issue Station)

중앙대학교 2024년 1학기 소프트웨어공학 Term 프로젝트  
2분반 11팀 이슈역 통합 Repository

<br><br>

## 🎯Details
<details>
<summary>주요 기능</summary>

- 계정 추가, 로그인(토큰 활용)
- 이슈 검색
- 이슈 등록
- 이슈 코멘트 추가
- 이슈 상세 정보 확인
- 이슈 배정을 포함한 이슈 상태 변경
- 이슈 통계 분석
- **Assignee 자동 추천**
- **Private, Public 속성으로 이슈 및 프로젝트 접근 관리**
</details>
